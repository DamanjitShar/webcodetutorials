HTML Tables.

  • HTML tables allow web developers to arrange data into rows and columns.

  • Each table cell is defined by a <td> and a </td> tag.

  • th stands for table heading.

  • tr stands for table table row.

  • td stands for table data.

    • <table>
    < tr>
    <th> Heading </th>
    </tr>
    <tr>
    <td> Data1 </td>
    </tr>
    < tr>
    <td> Data2 ></td>
    </tr>
    </table>
